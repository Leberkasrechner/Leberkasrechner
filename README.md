# Installation
To run this code, you will need:
1. A Webserver
2. PHP
3. A MySQL-Instance
4. The node modules given in the ```package.json``` file:

        npm install

5. A PHP package manager, preferably Composer. Requirements are listet in ```composer.json```. To install the requirements, run

        composer install
